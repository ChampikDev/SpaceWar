SPACE WAR
ver. 1.0.0

## Requirements

- Python 3.11 or newer
- pygame

## Start the game

From the project folder, create a virtual environment:

```powershell
python -m venv .venv
```

Activate it:

```powershell
.\.venv\Scripts\Activate.ps1
```

Install pygame:

```powershell
python -m pip install pygame
```

Run the game:

```powershell
python main.py
```

If `python` opens the Microsoft Store or does not work, run the commands with the full path to your installed Python executable instead.
